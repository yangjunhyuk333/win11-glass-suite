# Win11 Glass Suite 🌌

Win11 Glass Suite는 Windows 11의 유려한 Glassmorphism 디자인을 품은 미디어 컨트롤 및 시스템 모니터링 데스크탑 애플리케이션입니다. 사용자 인터페이스의 아름다움과 실용성을 동시에 추구합니다.

## ✨ 주요 기능

### 🎵 스마트 미디어 컨트롤
- 현재 재생 중인 음악 정보(제목, 아티스트)를 실시간으로 표시합니다.
- 앨범 아트를 유려한 블러 효과와 함께 시각화합니다.
- 재생/점지, 다음 곡, 이전 곡 제어 기능을 제공합니다.
- **Spotify** 및 다양한 윈도우 미디어 세션과 호환됩니다.

## 🚀 시작하기

### 실행 방법
1. [Releases](https://github.com/yangjunhyuk333/win11-glass-suite/releases) 페이지에서 최신 패키지를 다운로드합니다.
2. 압축을 해제한 후 `Win11 Glass Suite.exe`를 실행합니다.

### 개발 환경 설정
```bash
# 저장소 복제
git clone https://github.com/yangjunhyuk333/win11-glass-suite.git

# 의존성 설치
npm install

# 개발 모드 실행
npm run dev

# 빌드
npm run build
```

## 🛠️ 기술 스택
- **Core**: Electron, Vite
- **Frontend**: React, Framer Motion, Styled-components
- **Backend**: Node.js, PowerShell (System Interop)
- **Styling**: Glassmorphism CSS
